## 基于模式自动归纳的开放域事件抽取 的训练过程

### 一、基于主题模型和预训练模型的事件类型归纳 NVLDA-BERT(2周)
(一)、识别出文本中的候选事件触发词.

(二)、基于模型对候选事件触发词进行聚类，将聚类得到的候选事件触发词集合作为归纳出的事件类型表示。 

### 二、基于事件触发词聚类的语料检索（1周）
基于候选事件触发词聚类集合，采用信息检索的方法从原始开放域新闻语料库中检索与候选事件触发词聚类集合打分较高的相关文档作为对应此触发词集合的特定类型事件语料。
    文档与触发词聚类集合𝑐的打分通过两个指标：
    

    1.词匹配度
        
    2.词覆盖度


### 三、基于图表示学习的事件论元角色归纳 PinSAGE（2周）
(一)、构建论元角色关系图。在每类事件语料文本中抽取出事件论元角色相关的实体及其类别、实体与触发词间的句法关系、论元角色表达式等信息，基于这些相关元素之间的句法、语义、共现等关系构建论元角色关系图。

(二)、基于图卷积神经网络图表示学习算法计算论元角色表达式的表示。

(三)、最后对论元角色表达式进行聚类和筛选归纳得到事件论元角色。



事件抽取例子：

```text
我打个比方，输入是这样一个句子：
李帅于2022年12月4日在中国四川泸州的2022红旗杯红警比赛中击败了零八，夺得了红警世界冠军！
（以下用 “句子” 替代上述句子）
句子 → 触发词提取模型（BERT） → 触发词集合['击败','夺得'] → 事件类型归纳模型（LDA） → 事件类型集合['竞赛行为-胜负','竞赛行为-夺冠']
根据事件类型集合 以获取 对应论元角色集合，其对应关系在对PinSAGE模型训练后得出。
假设已经训练好对应关系为：
{'事件类型':'竞赛行为-胜负','论元角色':[赛事名称，胜方，败方，比赛时间]}
{'事件类型':'竞赛行为-夺冠','论元角色':[赛事名称，冠军，比赛时间]}

遍历上述关系：
句子 → PinSAGE-关系[0] → {'事件类型':'竞赛行为-胜负','论元':['2022红旗杯红警比赛','李帅','零八','2022年12月4日']}
句子 → PinSAGE-关系[1] → {'事件类型':'竞赛行为-夺冠','论元':['2022红旗杯红警比赛','李帅','2022年12月4日']}
如果将上述输出分别命名为类型1事件和类型2事件，则最终输入和输入如下：

句子 → 模型 → 输出[类型1事件，类型2事件]
```
