模板语法的学习

[Django遇到占位符会到views.py文件中去寻找相应模板来替换数据，浏览器读取到的数据则已是替换后的数据]

1.单独字符串输出

{{ n1 }}



{#1.列表#}

2.列表输出

{{ n2 }}

法1：直接索引，用”.“表示

{#法1：直接索引，用”.“表示#}
{{ n2.0 }}
{{ n2.1 }}
{{ n2.2 }}

法2：for循环索引

{#法2：for循环索引#}
{% for item in n2 %} {{ item }} {% endfor %}



{#2.字典#}

3.字典输出

{{ n3 }}

法1：直接索引，用键值索引

{#法1：直接索引，用键值索引#}
{{ n3.name }}
{{ n3.salary }}
{{ n3.role }}

法2：for循环索引

{#法2：for循环索引#}

    (1)”.keys“-->循环得到键值

    {#[”.keys“-->循环得到键值]#} {% for item in n3.keys %}
  • {{ item }}
    • {% endfor %}

    (2)”.values“-->循环得到值

    {#[”.values“-->循环得到值]#} {% for item in n3.values %}
  • {{ item }}
    • {% endfor %}

    (3)”.items“-->同时取键值与取值

    {#[”.items“-->同时取键值与取值]#} {% for k,v in n3.items %}
  • {{ k }}={{ v }}
    • {% endfor %}



{#3.列表嵌套字典#}

4.列表嵌套字典输出

{{ n4 }}

法1：直接索引

(1)只用”.“索引至相应字典

{#法1：直接索引，用键值索引#}
{{ n4.0 }}
{{ n4.1 }}
{{ n4.2 }}

(2)用”.“外加相应键值索引至相应字典中的特定值

{{ n4.0.name }}
{{ n4.1.salary }}
{{ n4.2.role }}

法2：for循环索引

{#法2：for循环索引#}
    {#[”.keys“-->循环得到键值]#} {% for item in n4 %}
  • {{ item.name }} {{ item.salary }} {{ item.role }}
    • {% endfor %}



{#条件语句#}

条件语句[支持if，elif，else]

{% if n1 == "韩超" %}
name是韩超，哒哒哒哒哒
name是韩超，哒哒哒哒哒
{% elif n1 == "xxx" %}
name不是韩超，哔哔哔哔哔
{% else %}
name不是韩超，嘟嘟嘟嘟嘟
{% endif %}
